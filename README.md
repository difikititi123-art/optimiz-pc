# 🚀 Windows 10 Optimization & Tweaks

Сборник полезных материалов, конфигураций и скриптов, вдохновленный ведущими проектами по глубокой оптимизации **Windows 10** для геймеров и энтузиастов. 

Главная цель — максимальный FPS, минимальный input lag (задержка ввода) и полное удаление системного мусора (телеметрии, бэкграунд-процессов).

---

## 🔍 Как узнать свою версию и сборку Windows 10?

Перед установкой обязательно проверьте, совпадает ли ваша сборка с требованиями оптимизаторов. Сделать это можно двумя простыми способами:

### Способ 1: Через команду `winver` (Самый быстрый)
1. Нажмите комбинацию клавиш **Win + R** на клавиатуре (откроется окно «Выполнить»).
2. Введите команду `winver` и нажмите **Enter**.
3. Откроется небольшое окно, где будут указаны **Версия** (например, 22H2) и **Сборка ОС** (например, 19045.xxxx).

### Способ 2: Через Параметры системы
1. Откройте **Параметры** (комбинация клавиш **Win + I**).
2. Перейдите в раздел **Система** -> в самом низу выберите вкладку **О программе**.
3. Прокрутите вниз до блока **«Характеристики Windows»**. Там будут указаны строка *Выпуск* (Windows 10 Pro/Home), *Версия* и *Сборка ОС*.

## ⚡ Что дают эти оптимизации?

| Параметр | Эффект после оптимизации |
| :--- | :--- |
| **Input Lag** | Минимальная задержка мыши и клавиатуры за счет оптимизации прерываний (MSI Mode) и таймеров. |
| **Процессы** | Снижение количества фоновых процессов Windows (со 150+ до 30-40). |
| **Память** | Освобождение оперативной памяти и отключение тяжелой телеметрии Microsoft. |
| **FPS** | Стабильный фреймрейт и сглаживание графиков времени кадра (Frame Time) в соревновательных играх. |

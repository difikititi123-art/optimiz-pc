# 🚀 Windows Optimization & Tweaks

Сборник полезных материалов, конфигураций и скриптов, вдохновленный ведущими проектами по глубокой оптимизации Windows для геймеров и энтузиастов. 

Главная цель — максимальный FPS, минимальный input lag (задержка ввода) и полное удаление системного мусора (телеметрии, бэкграунд-процессов).

---

## 🛠️ Первоисточники и Проекты-Вдохновители

Вся база и лучшие практики оптимизации систем взяты у официальных разработчиков:

*   **[AtlasOS (GitHub)](https://github.com/Atlas-OS/Atlas)** — Официальный open-source репозиторий проекта Atlas. Прозрачный код, глубокий тюнинг планировщика Windows, отключение лишних служб и минимизация задержек.
*   **[AtlasOS (Официальный сайт)](https://atlasos.net/)** — Главный ресурс проекта Atlas, где можно скачать актуальные конфигурации (Playbooks) для безопасного и быстрого изменения системы.
*   **[ReviOS / Revision](https://revi.cc/)** — Известный проект по созданию кастомных, стабильных и максимально облегченных систем (ReviOS). Отличается собственным удобным инструментом управления компонентами системы и высокой стабильностью в играх.

---

## ⚡ Что дают эти оптимизации?

| Параметр | Эффект после оптимизации |
| :--- | :--- |
| **Input Lag** | Минимальная задержка мыши и клавиатуры за счет оптимизации прерываний (MSI Mode) и таймеров. |
| **Процессы** | Снижение количества фоновых процессов Windows (со 150+ до 30-40). |
| **Память** | Освобождение оперативной памяти и отключение тяжелой телеметрии Microsoft. |
| **FPS** | Стабильный фреймрейт и сглаживание графиков времени кадра (Frame Time) в соревновательных играх. |
